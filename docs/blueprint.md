# **App Name**: NutriTrack Pro

## Core Features:

- User Profile Creation: Allow users to create profiles with detailed anthropometric data (age, sex, height, weight, activity level) to calculate personalized nutritional status.
- Nutritional Status Calculation: Automatically calculate and display nutritional status using user data.
- Calorie Requirement Calculation: Determine daily calorie and macronutrient needs based on user profile and goals using a rule-based tool.
- Dietary Logging: Enable users to log their daily food intake.
- Food Composition Database: Integrate with the TKPI food composition database to access nutritional information for logged foods. Access url: TKPI
- Nutrient Analysis: Analyze the user's dietary logs to display total calories, macros, and micros consumed based on the TKPI database.

## Style Guidelines:

- Primary color: Light green (#A7D9A9) to evoke a sense of health and freshness.
- Background color: Very light desaturated green (#F5F9F5).
- Accent color: A slightly more saturated and bright yellowish green (#C9E886) for key actions and highlights.
- Font pairing: 'Inter' (sans-serif) for body text and 'PT Sans' (sans-serif) for headers.
- Use clean, simple icons to represent different food groups and nutrients.
- Clean, minimalist layout with a focus on readability and usability.
- Subtle transitions and animations to provide feedback and enhance the user experience.
